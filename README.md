## My entry for the fabulous [Advent of Code 2023](https://adventofcode.com/2023/about)

### Requirements:
- PHP 8.2
- Composer

### Installation
1. `composer install`
2. Add your adventofcode.com session cookie contents to `app.ini` (required to automatically download input files)

### Run
`composer day [number]`
